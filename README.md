Tested on lts/hydrogen

Install deps

```sh
yarn
```

Run UI

```sh
cd apps/ui;yarn dev
```

Run BE

```sh
cd apps/api;yarn build;yarn start
```

# Assumptions

- Will add cache, short on time ATM
- using wallet connect project id (should make this private repo)
- okay to use UI libs like radix UI
- only mainnet tokens
